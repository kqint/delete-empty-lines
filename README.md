# Obsidian 空行删除插件

## 📝 简介

用于删除或压缩笔记中的空行。支持全文处理和选中区域处理，可自定义保留的空行数量。

## 📖 使用方法

### 命令面板（可设置快捷键）

按 `Ctrl/Cmd+P`，搜索“压缩空行”即可看到两个动态命令：

- `压缩空行（全文，保留 X 行）`
- `压缩空行（选中区域，保留 X 行）`

命令名称中的数字会随着设置自动更新。

### 右键菜单

在编辑器中右键点击：

- **有选中文本时**：显示“压缩选中区域空行”选项
- **无选中文本时**：显示“压缩全文空行”选项

## 🚀 安装方法

### 手动安装

1. 下载 `main.js` 和 `manifest.json` 文件
2. 在 Obsidian 库中创建文件夹：`.obsidian/plugins/delete-empty-lines/`
3. 将下载的文件放入该文件夹
4. 在 Obsidian 设置中启用插件

## ⚙️ 配置说明

### 1. 删除只包含空格和制表符的空行

- **开启**：只包含空格和制表符的行被视为空行，会被删除
- **关闭**：只包含空格和制表符的行被视为非空行，会被保留

*示例：一行中只有5个空格，没有其他字符。开启该选项时会被删除，关闭时会被保留。*

### 2. 全文默认最大连续空行数

处理整个文档时，保留的最大连续空行数

- 设置为 `0`：删除所有空行
- 设置为 `1`：最多保留1个连续空行
- 设置为 `2`：最多保留2个连续空行，以此类推

### 3. 选中区域默认最大连续空行数

处理选中文本时，保留的最大连续空行数（规则同上）
